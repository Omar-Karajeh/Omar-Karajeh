### 🔴 [Active Directory Detection Lab](https://github.com/Omar-Karajeh/Active-Directory-Detection-Lab)
> Self-hosted AD attack detection lab — Windows Server 2022 + Windows 10 + Splunk

Built a complete adversary simulation and detection environment:
- **Kerberoasting** (T1558.003) — Detected via Event ID 4769 + RC4 Encryption Type `0x17`
- **Pass-the-Hash** (T1550.002) — Detected via Event ID 4624 Logon Type 9 + `seclogo`
- **DCSync** (T1003.006) — Detected via Event ID 4662 + DS-Replication GUIDs
- Includes **SIGMA Rules**, **Splunk SPL Queries**, **Multi-Source Correlation** across Sysmon and Windows Security logs
- Full **MITRE ATT&CK mapping** and **Attack Timelines** documented
